\# 随身media（暂定名）



一个使用 WPF + C# 编写的本地音乐播放器。



最初为了复习 C# 和练习项目结构，后来变成了长期自用的播放器。



主打：本地音乐播放、多列表缓存、快速切换、轻量个人使用、插U盘即可运行。



\# 项目特点



\## 自动缓存系统

启动时自动扫描音乐目录，提前缓存歌曲名称、文件路径、歌曲时长，避免后续播放时频繁读取文件导致卡顿。歌曲时长使用异步读取，防止 UI 卡死。



\## 多列表系统

支持多个独立歌单缓存：highmusician、awakemusician、anothermusician，方便根据不同状态快速切换歌曲。



\## 播放模式

支持顺序播放、随机播放、单曲循环。



\## UI同步

切歌时统一更新标题、进度条、当前歌曲、播放状态。核心逻辑集中在 `musicchange()` 方法中。



\# 项目目标

不追求商业化、完美架构、工业级规范。更像一个个人开发者真正开始“做软件”的阶段记录。



目前用于：复习 C#、学习 WPF、训练项目结构能力、练习缓存与状态管理、为后续 Unity 开发打基础。



\# 已知问题

\- 路径部分仍然存在写死情况

\- UI 仍有优化空间

\- 部分逻辑后期编写较为随性

\- 暂未完全模块化



\# 未来计划

更好的 UI、U盘完全自动适配、更稳定的播放逻辑、更多播放列表、壁纸/主题、更清晰的结构拆分。



\# 作者的一点废话

这个播放器最开始只是个练习项目。但后来发现：真正让人舒服的软件，不一定非得多复杂。有时候一个顺手的工具、一段喜欢的音乐、一个深夜还能正常运行的小程序，就已经足够好了。
